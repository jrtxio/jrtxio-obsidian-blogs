2026年3月9日，腾讯正式推出全场景 AI 智能体 **WorkBuddy**。

这个被誉为"腾讯版小龙虾"的工具，一出场就惊艳了整个科技圈。最让人惊喜的是：**从下载安装到连接企业微信，最快只需 1 分钟**。

## WorkBuddy 是什么？

简单来说，WorkBuddy 是一个**能听懂人话、带脑子思考、真能操作电脑**的桌面 AI 智能体。

它不是那种只会聊天的 AI 助手，而是能够**帮你真正干活**的工作伙伴：

- 你在手机上发一句语音指令
- 它在办公电脑上自动执行任务
- 最后交付给你一个**可直接验收的成果**

比如，你在通勤路上说一声"帮我整理一下上周的销售数据并发送邮件"，等你到公司，报告已经发到你邮箱了。

## 核心亮点：为什么这么好用？

### 1. 开箱即用，零学习成本

传统的 AI 工具往往需要复杂的云端部署，配置起来让人头秃。WorkBuddy 彻底改变了这一点：

```
下载 → 安装 → 输入指令 → 开始工作
```

就这么简单，四步搞定。

如果你是腾讯企业微信用户，还能通过远程配置功能，**1 分钟内完成连接**，直接用手机"遥控"电脑上的 WorkBuddy 干活。

### 2. 兼容 OpenClaw 技能，生态丰富

WorkBuddy 完全兼容 OpenClaw（俗称"小龙虾"）的技能。这意味着：

- 你可以直接使用 OpenClaw 生态中的所有 Skills
- 不需要重复学习新的技能体系
- 社区资源拿来即用

目前内置超过 **20 种技能包**，包括：
- 文档处理
- 数据分析
- 内容创作
- 自动化报表
- 海报生成

### 3. 多模型无缝切换

WorkBuddy 国内版支持多种主流大模型的无缝切换：

- **混元**（腾讯自研）
- **DeepSeek**
- **GLM**
- **Kimi**
- **MiniMax**

你可以根据任务特点选择最合适的模型，或者让 WorkBuddy 自动选择。灵活多变，满足不同场景需求。

### 4. 多窗口、多 Agent 并行工作

这是 WorkBuddy 的杀手级功能。

面对复杂任务时，它可以：
- **动态拆解任务**：将大任务分解为多个子任务
- **多 Agent 并行**：同时调度多个 AI 助手协同工作
- **多窗口展示**：让你实时掌控每个任务的进度

想象一下，你要准备一个项目汇报：
```
Agent 1：整理项目数据
Agent 2：生成数据图表
Agent 3：撰写汇报文案
Agent 4：制作 PPT
```

四个 AI 同时开工，几分钟完成你几小时的工作量。

### 5. 深度集成办公生态

WorkBuddy 不只是一个独立的工具，它能与你常用的办公软件无缝协作：

- **企业微信**：远程控制、消息推送
- **QQ**：快速分享任务结果
- **飞书**：集成文档协作
- **钉钉**：消息通知和工作流

你不需要切换应用，一切都在熟悉的生态中完成。

## 如何开始使用？

### 第一步：下载安装

访问 WorkBuddy 官网：https://www.codebuddy.cn/work/

下载对应系统的安装包，双击安装即可。

### 第二步：配置连接

如果是企业微信用户：
1. 打开 WorkBuddy
2. 选择企业微信配置
3. 扫码授权
4. 完成（约 1 分钟）

如果不是企业微信用户：
1. 直接在 WorkBuddy 界面输入指令
2. 即可开始使用

### 第三步：选择技能和模型

首次使用时，建议：
- 导入常用的 Skills 技能包
- 选择一个熟悉的大模型开始
- 尝试简单任务熟悉操作

### 第四步：开始干活

输入你的第一个指令：

```
帮我整理一下这个文件夹里的文档
```

然后，就可以坐等成果了。
