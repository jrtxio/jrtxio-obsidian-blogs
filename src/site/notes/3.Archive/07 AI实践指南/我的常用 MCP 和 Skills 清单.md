#ai

## MCP

| 名称                  | 用途说明                                                                                                                   |
| ------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| context7            | 获取和查询任何编程库或框架的最新文档和代码示例。支持实时文档检索，帮助开发者了解最新的 API、最佳实践和代码示例                                                              |
| sequential thinking | 动态和反思性问题解决工具。通过灵活的思考过程分析问题，可以适应和调整，支持分支、修正和重新分析，适用于复杂问题的多步骤解决方案                                                        |
| pencil              | 设计工具的 MCP 服务。基于 Model Context Protocol (MCP) 协议，提供无限画布设计功能，支持 AI 助手直接调用和交互设计文件，实现设计到代码的无缝转换，集成于开发环境中如 VS Code 和 Cursor |

## Skills

| 名称              | 用途说明                                                                                                                                                                                                           |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| frontend-design | 创建独特、生产级的前端界面，设计质量高。适用于构建网页组件、页面、海报或应用程序（如网站、落地页、仪表板、React 组件、HTML/CSS 布局等）。生成创意、精美的代码和 UI 设计，避免通用的 AI 审美                                                                                                      |
| ui-ux-pro-max   | UI/UX 设计智能助手。提供 50 种风格、21 种调色板、50 种字体搭配、20 种图表类型、9 种技术栈（React、Next.js、Vue、Svelte、SwiftUI、React Native、Flutter、Tailwind、shadcn/ui）。支持规划、构建、设计、实现、审查、修复、优化和增强 UI/UX 代码。适用于网站、落地页、仪表板、管理面板、电商、SaaS、作品集、博客、移动应用等项目 |
| mcp-builder     | 构建高质量 MCP (Model Context Protocol) 服务器的指南。使 LLM 能够通过精心设计的工具与外部服务交互。用于集成外部 API 或服务，支持 Python (FastMCP) 或 Node/TypeScript (MCP SDK)                                                                              |
| skill-creator   | 创建有效技能的指南。用于创建新技能或更新现有技能，扩展 Claude 的能力，包括专业知识、工作流或工具集成                                                                                                                                                         |
